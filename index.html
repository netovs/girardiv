Ejemplo de girar divs

En este ejemplo aprenderás a girar un div para realizar diversos efectos visuales, en el ejemplo muestro como girar una llanta (en México le decimos llanta a los neumáticos)

Función que detecta las propiedades para girar el div en cada navegador... obvio en Internet Explorer 8 no funciona...

`


  function getTransformProperty(element) { 
  
  var properties = [

    'transform',

    'WebkitTransform',

    'msTransform',

    'MozTransform',

    'OTransform'

];

var p;

while (p = properties.shift()) {

    if (typeof element.style[p] != 'undefined') {

        return p;

    }

}

return false;
}

`

Una vez que obtengas las propiedades para transformar los div por cada navegador viene el llamado a la rotación del elemento utilizando jQuery...

`


  
$( document ).ready(function() {

JQuery.fn.rotate = function() {


var div = document.getElementById('rotate');

var property = getTransformProperty(div);

if (property) {
  
  var d = 0;

setInterval(

    function () {

        div.style[property] = 'rotate(' + (d++ % 360) + 'deg)';

    },

    1

);
}

return $(this);

};

$(".rotate").click(function() {

$(this).rotate(); 
});

});










`
